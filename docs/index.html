How to Install

Chrome Web Store (Recommended - Coming Soon)

The extension is currently under review and will be available on the Chrome Web Store soon. This is the easiest and most secure way to install.

Manual Installation (For Now)

You can install the extension manually while we're waiting for Chrome Web Store approval:

  1. Download the latest release from GitHub Releases
  2. Extract the zip file to a folder on your computer
  3. Open Chrome and go to chrome://extensions/
  4. Enable "Developer mode" using the toggle in the top right
  5. Click "Load unpacked" and select the extracted folder
  6. Open a new tab and enjoy Horizon!
‚ö° Note: Once installed, you can use Horizon completely offline. Sign-in is optional and only required for cloud sync across devices.
